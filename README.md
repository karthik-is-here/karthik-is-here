## featured project

### 🗺️ kochi traffic intelligence
> predicting congestion & accident risk across 58 roads in kochi using real-time data

- end-to-end ML pipeline: data collection → feature engineering → XGBoost → dashboard
- live data collected via **Termux on Android**
- interactive map dashboard built with **Streamlit + Folium**
- covers temporal patterns, weather, road type, and historical incident data
